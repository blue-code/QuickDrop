# QuickDrop Clone

macOS용 프로그램입니다. 화면 상단에서 빠르게 파일과 폴더에 접근할 수 있는 생산성 도구입니다.

## 주요 기능

- 🖱️ **자동 표시/숨김**: 마우스를 화면 상단으로 이동하면 자동으로 창이 나타납니다
- 📁 **다중 워크스페이스**: 여러 프로젝트 폴더를 워크스페이스로 관리
- 🔝 **최상위 창**: 다른 모든 창 위에 표시되어 항상 접근 가능
- 📌 **고정 기능**: 창을 고정하여 항상 표시되도록 설정
- 🎯 **드래그 앤 드롭**: 파일을 다른 앱으로 쉽게 드래그
- 🔍 **컨텍스트 메뉴**: 파일 열기, Finder에서 보기 등
- 💾 **자동 저장**: 워크스페이스 설정이 자동으로 저장됨

## 시스템 요구사항

- macOS 10.15 (Catalina) 이상
- Python 3.8 이상

## 개발 환경 설정

### 1. 저장소 클론 및 의존성 설치

```bash
# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화
source venv/bin/activate

# 의존성 설치
pip install -r requirements.txt
```

### 2. 개발 모드 실행

```bash
python main.py
```

## 빌드 및 배포

### DMG 파일 빌드

프로젝트에 포함된 빌드 스크립트를 사용하여 자동으로 빌드할 수 있습니다:

```bash
# 실행 권한 부여
chmod +x build.sh

# 빌드 실행
./build.sh
```

빌드 스크립트는 다음 작업을 수행합니다:
1. 가상환경 생성 및 의존성 설치
2. 기존 빌드 파일 정리
3. .app 번들 생성
4. DMG 파일 생성

### 수동 빌드

원하는 경우 수동으로 빌드할 수도 있습니다:

```bash
# .app 번들 생성
python setup.py py2app

# DMG 생성 (선택사항)
hdiutil create -volname "QuickDrop" -srcfolder dist -ov -format UDZO FolderHub.dmg
```

## 설치 방법

1. 생성된 `FolderHub-1.0.0.dmg` 파일을 더블클릭
2. `QuickDrop.app`을 Applications 폴더로 드래그
3. Applications 폴더에서 QuickDrop 실행

## 사용 방법

### 기본 사용법

1. **앱 실행**: Applications 폴더에서 QuickDrop를 실행
2. **창 표시**: 마우스 커서를 화면 상단으로 이동
3. **워크스페이스 추가**: "+ 워크스페이스" 버튼 클릭
4. **파일 접근**: 탭을 전환하여 다른 워크스페이스의 파일 접근

### 주요 단축키 및 기능

- **마우스를 상단으로**: 창 자동 표시
- **마우스를 밖으로**: 창 자동 숨김 (고정되지 않은 경우)
- **📌 버튼**: 창 고정/해제
- **우클릭**: 컨텍스트 메뉴 (열기, Finder에서 보기, 새로고침)
- **드래그**: 파일을 다른 앱으로 드래그 앤 드롭

### 워크스페이스 관리

1. **추가**: "+ 워크스페이스" 버튼 → 이름과 경로 입력
2. **제거**: 탭의 ✕ 버튼 클릭
3. **전환**: 탭 클릭 또는 마우스 오버

## 설정 파일

설정은 자동으로 저장됩니다:
- 위치: `~/.folder_hub_config.json`
- 내용: 워크스페이스 이름과 경로

## 문제 해결

### 앱이 실행되지 않는 경우

macOS에서 인증되지 않은 개발자의 앱을 차단할 수 있습니다:

1. 시스템 환경설정 → 보안 및 개인 정보 보호
2. "확인 없이 열기" 버튼 클릭

또는 터미널에서:
```bash
xattr -cr /Applications/Folder\ Hub.app
```

### 권한 문제

파일 시스템 접근 권한이 필요할 수 있습니다:

1. 시스템 환경설정 → 보안 및 개인 정보 보호 → 개인 정보 보호
2. 파일 및 폴더 → QuickDrop에 권한 부여

## 개발 정보

### 프로젝트 구조

```
folder-hub/
├── main.py              # 메인 애플리케이션 코드
├── setup.py             # py2app 빌드 설정
├── requirements.txt     # Python 의존성
├── build.sh            # 자동 빌드 스크립트
└── README.md           # 이 파일
```

### 사용된 기술

- **PyQt6**: GUI 프레임워크
- **py2app**: macOS 앱 번들 생성
- **hdiutil**: DMG 파일 생성

### 커스터마이징

#### 스타일 변경
`main.py`의 `setStyleSheet()` 부분을 수정하여 색상과 디자인을 변경할 수 있습니다.

#### 아이콘 추가
1. 앱 아이콘 파일 준비 (`.icns` 형식)
2. `setup.py`의 `iconfile` 옵션에 경로 지정:
```python
'iconfile': 'icon.icns',
```

#### Dock 아이콘 숨김
`setup.py`의 plist 설정에서:
```python
'LSUIElement': True,  # Dock 아이콘 숨김
```
